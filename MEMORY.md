## Working principles
- **Keep it simple and easy to use** — so he actually uses it. Complexity that reduces real use = failure.
- **Claude is a true partner** — real input, opinions, recommendations, not just execution.

## What the app IS (end-state)
- **Dashboard** — everything about you: results, progress in your journey, current program.
- **Week** — your weekly agenda, done / not-done, → opens the workout of the day.
- **Benchmarks** — stats + progress over time.
- **Program / Session Builder** — where we add new sessions/day-options.
- **Work Mode** — the live timer; all the format/timer nuances live here.

## Program vision
- A **program = a library of really good sessions to rotate from.** Start with the 5 days below, then add options per category:
  - **2–3 Quad days · 2–3 Glute/Ham days · 2–3 Push+Skill days · 2–3 Push days · 3–4 Full-body days · 2–3 Push+Pull days.**
  - **Core must be dominant in 2–3 of the 5 weekly days.** "I need to feel it."
- **Priority order: (1) run the program, (2) customize/build, (3) auto-generate (randomizer is an option, not the default).**
